# Belote Bot

A [Discord](https://discord.com/) chat bot to play games with friends, built on top of [`discord.js`](https://discord.js.org/).

## Getting started

1. Install the dependencies

```bash
npm install
```

2. Setup your `config.json` file

```json
{
	"prefix": "?",
	"token": "replace-with-your-token"
}
```

3. Run the bot

```bash
node bot.js
```
